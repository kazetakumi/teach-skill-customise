# Teaching Notes

## Teaching Method
- **Follow [TEACHING-METHOD.md](TEACHING-METHOD.md) for every lesson.** Musk's
  first-principles approach: reasoning from fundamental truths instead of
  inherited convention, and building each slice through one ordered algorithm —
  question the requirement, delete, simplify, accelerate, automate. This governs
  *how* I'm taught, regardless of topic.
- **Ground facts in sources, not memory.** Derive mechanisms from scratch, but
  gather every external fact, dataset, number, and definition into `RESOURCES.md`
  from trusted sources first, and cite them in the lesson. Never invent facts from
  parametric memory.

## Lesson Output
- **Follow [LESSON-FORMAT.md](LESSON-FORMAT.md): HTML is the medium I teach
  through.** Every lesson is one HTML file, code runs live in a throwaway scratch
  surface, every number and plot on the page comes from a real run.

## Style
- **Style every HTML lesson per [STYLE.md](STYLE.md).** One editorial, warm,
  code-forward look, clean and Karpathy-minimal. Canonical reference:
  `_style-examples/Bprime-reference.html`.

Retention/spaced-review is deliberately unwired — no review step runs.

## User Preferences
- **First-principles learner, Musk-flavored.** I question the requirement before
  accepting it, not just derive the mechanism from primitives. If a "have to"
  doesn't trace to a fundamental truth, I want it deleted, not built around.
- **Plain language, physical intuition for math.** Explain in simple language. Every
  equation should make physical sense before it's symbols — show me what's actually
  happening (a picture, a concrete number, a cause-and-effect) and only then attach
  the formula and the term for it.
- **Mental-model first.** The goal of every lesson is a clear mental model I could
  rebuild from a blank page, not just a working answer.
- **Hands on the keyboard.** I write the naive version and predict outputs before
  they're revealed — I don't just watch.
- **Teachability is the success test.** I should be able to explain the concept to
  anyone, in simple language, from first principles. If I can't teach it simply,
  the lesson isn't done.
